# How to Run Cat Shelter Workshop

## 1. Install dependencies

```bash
npm install
```

## 2. Start the server

```bash
npm start
```

## 3. Open the application

```txt
http://localhost:3001
```

## Notes

The application uses JSON files as a simple database:

```txt
data/cats.json
data/breeds.json
```

Uploaded cat images are saved in:

```txt
content/images/
```

If node_modules is missing, run npm install again.
